 Field Name     | Type   | Description
---------------:| ------ | ------------------------
itemid          | Number | 物品编号 `yyyy\d{5}`
name            | String | 物品简略名称
description     | String | 物品详细描述，可以添加注意事项
count           | Number | 剩余数量
tags            | String | 标签，用于分类
repoid          | String | 仓库编号，详见下

仓库    |详情 
--------|-------
一号仓库|1、2、3货架
二号仓库|1、2、3、4、5、6货架，以及用于放置旗帜的分区，例如某旗帜可表示为二号仓库A1
三号仓库|无特别货架
四号仓库|1、2、3货架，每个货架有A、B、C……分区
